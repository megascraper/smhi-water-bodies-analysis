# SMHI Water Bodies Analysis
Public data downloaded from https://www.smhi.se/
A geospatial Python project for analyzing and visualizing Swedish water bodies using shapefile data.


- Loads and inspects geospatial water body data
- Calculates area in km²
- Shows category distributions
- Identifies the 10 largest water bodies
- Creates static thematic maps
- Exports an interactive Folium web map


- Python
- GeoPandas
- Matplotlib
- NumPy
- Folium

pip install -r requirements.txt
